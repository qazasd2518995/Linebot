How to Create a LINE Channel

Step 1: Go to LINE Developer Console

Visit developers.line.biz/console and log in with your LINE account.

Step 2: Create a Provider

Click "Create" and enter a provider name (e.g., your name or "SLA Project").

Step 3: Create a Messaging API Channel

Click "Create a Messaging API channel" and fill in:

  • Channel name: Your bot's name (e.g., "Speaking Coach")
  • Channel description: Brief description
  • Category: Education
  • Subcategory: School

Step 4: Get Your Credentials

Channel Secret: Basic settings tab → Channel secret

Channel Access Token: Messaging API tab → Click "Issue" to generate

Step 5: Disable Auto-Reply

In Messaging API tab → Auto-reply messages → Click "Edit" → Turn OFF auto-reply
